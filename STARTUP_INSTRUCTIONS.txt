# PharmaSync Project Startup Instructions

This project consists of a Django REST API backend and a React (Vite) frontend.

## Prerequisites
- Python 3.10+
- Node.js (v18 or higher)
- PostgreSQL (ensure 'pharmacy_db' exists)

---

## Step 1: Start the Backend (Django)
1. Open a terminal in the root directory.
2. Activate environment: `.\venv\Scripts\activate`
3. Run: `python manage.py runserver 8000`

---

## Step 2: Start the Frontend (React)
1. Open a second terminal.
2. Run: `cd frontend`
3. Run: `npm run dev` (Available at http://localhost:5173)

---

## New Features Implemented
- **JWT Authentication:** Secure Login and Registration (Signup).
- **Automatic Stock Deduction:** FIFO logic automatically reduces stock during sales.
- **Full Inventory CRUD:** You can now Save, Update, and Delete medicines.
- **Advanced Billing:** Real-time cart management and stock validation.
- **UI/UX Polishing:** Fixed chart rendering issues and improved responsiveness.

---

## Authentication
- **Register:** Go to `/register` to create a new pharmacist account.
- **Login:** Use your credentials at `/login` to access the dashboard.
